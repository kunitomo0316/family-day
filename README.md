# 社内 SNS α 版 フロントエンド

## Git Clone

```
git clone https://github.com/kunitomo0316/family-day.git
```

## プロジェクト実行

### ディレクトリ移動

```
cd family-day
```

### モジュールのインストール

```
npm install
```

### サーバー立てる

```
npm run start
```

## 作成した画面

### 投稿画面

http://localhost:3000/Top

### 一覧画面（削除予定）

http://localhost:3000/Display

### その他のリンク

投稿画面に自動遷移するようになってます
